Installation

Installation of any phpBB theme is the same and the only differences lie in the name of the theme/template. In the below instructions we will use fisubsilver as an example - you should replace fisubsilver with the name of the template you are installing.

Uncompress the distribution archive (using such as Winzip) into your phpBB templates directory, making sure that the file and directory structure is retained, e.g. phpBB/templates/fisubsilver/

Log into the phpBB Administration Panel.

Click on Add under the Styles Administration title.

If all is present and correct you should see the fisubsilver name listed under Style and Template. Click on Install at the side of the fisubsilver name under the Action title.

The theme should then be available for you and your users to choose from.
